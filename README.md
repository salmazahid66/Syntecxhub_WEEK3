- Visualized survival patterns

### Key Insights

- First-class passengers survived more
- Children had better survival chances

- Analyzed content distribution by type and year
- Identified top genres and countries
- Visualized growth trends

### Key Insights
- Movies dominate Netflix content
- Significant growth after 2015
- USA is top content producer
